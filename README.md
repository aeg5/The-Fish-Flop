# The-Fish-Flop
This is a PCB for Hack Club Solder! It's a simple flip flop circuit that's fish shaped. The LED can be toggled with the set and reset buttons. My slack username is ag5.

## BOM
- 1 battery holder
- 1 LED
- 2 NPN transistors
- 2 330 resistors
- 2 1k resistors 
- 2 2k resistors
- 2 100k resistors
- 2 push buttons
